What is the performance impact of missing school?¶

In this notebook, I will attempt to identify the performance impact of students missing school. I will use public data sourced from the NJ DOE. As schools deal with Covid-19, there is a chance of students missing days of school as schools cycle through shut-downs, distant learning, and reopening. In doing this, there is a real risk of students seeing a reduction in performance. The focus will be on elementary schools in aim to set up our students for continued success in secondary education and beyond. These students are also particularly inexperienced with distance learning

def identify_nulls(df, cols_to_check):
    ''' Takes a dataframe and list of column names and returns a set of the non-numeric values in column '''
    null_vals = set()
    for col_name in cols_to_check:
        col_series = df[col_name]
        for value in col_series.values:
            if isinstance(value, Number) == False:
                if value.replace('.','').isdigit() == False:
                    null_vals.add(value)
    return null_vals
 
    
def replace_nulls(df, cols_to_replace, null_vals, replace_val):
    ''' Takes a dataframe, list of columns to replace, all the values to replace, and the replacement value and returns a modified dataframe'''
    for col_name in cols_to_replace:
        df[col_name].replace(list(null_vals), replace_val, inplace=True)

There are many different sheets in the source data file that we will be calling and selecting school data. Let us write another helper function that will reduce any future dataframe to only include those schools of interest.

In [160]:
def reduce_schools(df_reduce, df_schools = schools):
    ''' Take two dataframes: one with all source data that is to be reduced, and the second with schools to keep. 
        Merge dataframes so that only schools of interest are kept. '''
    # Count rows and columns
    r_i, c_i = df_reduce.shape
    
    # Create a unique identifier code for each school from component codes
    df_reduce['UniqueCode'] = df_reduce.CountyCode + df_reduce.DistrictCode + df_reduce.SchoolCode
    
    # Merge on the unique code, use inner merge to only keep shared keys (schools)
    df_reduce = df_reduce.merge(df_schools['UniqueCode'],how="inner",on="UniqueCode")
    
    # Count rows and columns
    r_o, c_o = df_reduce.shape
    
    print(f'Dataframe has been changed from {r_i} to {r_o} rows.')
    return df_reduce

Note that non-numeric values are identifed by:
* indicates that data was available for too few students to report the given information, or the data represents a small percentage of students. There may be some additional cases where the data was kept private because the data could be used to potentially identify individual students.
** indicates that data was not available for the minimum 20 students, the required number for a student group to be included in New Jersey’s Every Student Succeeds Act (ESSA) accountability system. This will only appear in sections of the report related to ESSA accountability.
N indicates that no data was available to report. This happens when there are no students enrolled in a particular student group or if no data was submitted by the district.
